# Raytracing in one weekend

This repo contains my work following the tutorial from the book ["Raytracing in one weekend"](https://raytracing.github.io/) by Peter Shirley. 

I'm trying to implement as much of it myself and before turning to the original code of the tutorial. I've found that to be a very efficient way to learn, as I can create a lot of opportinities for feedback.

Feel free to raise an issue if you have any comment, suggestion or correction if you see a mistake.
